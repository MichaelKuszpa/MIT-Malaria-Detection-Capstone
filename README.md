# MIT Applied Data Science Capstone – Malaria Detection Using Deep Learning

## Overview

This project was completed as part of the **MIT Professional Education Applied Data Science Program**.

The goal of this capstone project was to develop an end-to-end deep learning solution capable of classifying microscopic blood smear images as either **Parasitized** or **Uninfected** for malaria detection.

## Dataset

The project used a labeled image dataset containing over **27,000 microscopic blood smear images**.

Classification categories:

- **Parasitized**
- **Uninfected**

## Methods

This project included:

- Data preprocessing
- Exploratory data analysis
- Image normalization
- Convolutional Neural Network model development
- Transfer learning
- VGG16 model implementation
- Model evaluation and comparison

## Results

The final transfer learning model using **VGG16** achieved approximately **96% classification accuracy** in distinguishing parasitized and uninfected blood smear images.
